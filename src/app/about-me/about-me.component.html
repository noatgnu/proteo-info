About me

 Me

My work primarily focuses on development of bioinformatics tools and workflows for data processing and management of proteomics mass spectrometry data. This would involve creation of Python packages and modules that handle adapting and manipulating input and output of various workflows and databases and aggregating the results as final output. The input data could come from a number of search engine like Proteome Discoverer, Diann, Mascot, Maxquant, MSFragger... Following data processing and analysis is visualization of the results in javascript-based web applications where the interactive graphics could be shared with anyone without the need for installation of extra tools or software.
